# Guía Interactiva de CSS

Página de práctica y referencia visual con ejemplos de propiedades CSS
(colores, box model, box-sizing, botones), cada sección mostrando el
código junto a su resultado visual.

## Contenido actual

- Colores y fondos (con tooltip interactivo al hacer hover)
- Modelo de caja (Box Model) - diagrama visual de capas
- Comparación de box-sizing: content-box vs border-box
- Ejemplo de botón con padding y hover

## Cómo usar

Abre `index.html` en el navegador, o usa la extensión Live Server de
VS Code para verlo con recarga automática.
